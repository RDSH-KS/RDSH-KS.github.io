windows 10 必备软件

发表于 2020-03-05 更新于 2020-03-08 分类于 software

读书时，个人电脑还是稀罕物，整个学校也就是机房里有几十台电脑，大多装的是DOS系统，只有几台装的Windows（3.2？记不清了）。那时把喜欢折腾的软件和游戏存在软盘上，每次上机都随身带着，真是所谓的“便携软件”。后来，个人电脑越来越普及，windows越来越华丽。有段时间，下班熬夜组电脑、重装系统、折腾各类软件乐此不疲。现在，没有当初的熬夜的精力，对于新系统和软件好奇和兴奋也慢慢平静，更看重软件的实用、易用、稳定和经济。
因为使用环境和目的不同，“必备软件”这种说法有点绝对和夸张，千人千面，每台电脑上的软件集合也应该独一无二的。到后来终要明白：适合自己的需求才是最好的，能解决自己问题才是必备的。

系统 Windows 10 Pro

工作涉及工业装备的设计、制造，许多软件必须在windows平台运行，所以Windows 10是首选。对于一般用途，新购置的品牌电脑预装Win10 家庭版已经够用，有特殊需求，比如组策略权限、WSL之类就要选用专业版。专业版去微软家购买还是有门槛的，据说在马云家很容易解决。

文字处理 Office 365

Office 365应该是微软家应用最广最强的生产力工具。拼团购买家庭版，实在是比较经济靠谱的选择。 替代：WPS

病毒防护 火绒安全

火绒安全安静、简洁、方便。杀毒及防护能力个人没有足够的能力去测试和判断，但是与其它诸如360、XX电脑管家之类相比，让你感觉它没那么流氓。 替代：可以不装这类的软件，用系统自带 windows Defender。

视频播放 PotPlayer

PotPlayer是由原KMPlayer的制作者自己开发的免费软件。整个软件体积小，功能强，占用内存小，解码能力强，播放非常流畅，软件本身没有任何广告和流氓插件，可进行个性化配置适合自己喜好和需求。

音乐播放 Listen 1

Listen 1 是在Github上开源的音乐播放器，软件有Windowns, Mac, Linux 桌面版和 Chrome，Firefox插件版。软件简洁，优雅，可以搜索和播放来自网易云音乐，虾米，QQ音乐，酷狗音乐，酷我音乐等网站的音乐。

截图贴图 Snipaste

Snipaste 是一款简单但是功能强大的截图软件。F1键开始截图，F3键截图桌面置顶显示。软件截图定位精确，还可以将剪贴板里的文字或颜色信息转化为图片窗口，并进行缩放、旋转、透明等操纵。遗憾的是好像不支持滚动截图。专业版需要购买。替代：Win10自带的“截图和草图”软件，热键 Win+Shift+S 。

图片编辑 GIMP

GIMP 是一套跨平台，免费开源图像处理软件。没有Photoshop功能那么强大和方便，但是个人进行日常的图像编辑已经够用。

矢量作图 Inkscape

Inkscape 也是一款跨平台，免费开源软件。用于矢量图绘制和编辑。对于经常套用模板进行创作生产的人来说，无法与illustrator媲美，如果纯粹是自己设计、制作完全可以胜任。

机械三维 FreeCAD

FreeCAD 用不起UG、AUTOCAD、SolidWorks这些机械三维建模软件，可以尝试这个免费开源的参数化驱动三维建模软件。尽管是免费的，其功能可能会超出你的想象。

软件开发工具 Visual Studio Code

VSCode 微软号称“重新定义代码编辑”的跨平台开源开发工具。开源、免费、自定义配置、集成git、支持插件…… 程序开发、前端、后端、运营、还是内容创作，都拿的上手，火起来不是没有理由的。

解压缩软件 Bandizip

Bandizip 是一款简洁便捷高效的免费解压软件。原先没有广告，据说在7.0版本开始免费版本开始有功能限制和广告了。不看颜值的话，或许可以换用7-Zip，PeaZip？

图片浏览 Honeyview

Honeyview 是BandiZip同一家公司推出的图像浏览器。快捷、免费、无广告、可以查看EXIF信息，可以对图像进行批量的格式转换和调整。

词典 欧路词典

欧路词典 Win、MAC、安卓手机、苹果手机全平台，不同客户端，支持学习笔记、生词本同步。最主要的是词库资源丰富，随时可以根据自己需要安装适用的词库。

Everything

其它软件

  • TIM（替代QQ）的聊天交流软件
  • Wechat 微信似乎是这个时代中国人必备的手机沟通交流工具，PC端可方便文件传输，信息保存。
  • python 小巧、方便的程序开发语言，也是玩树莓派必备的东西。
